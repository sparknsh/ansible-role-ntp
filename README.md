## Role Variables

```yaml
ntp_enabled: True
ntp_timezone:

ntp_servers: []

ntp_restrict: []
```

#### Example

```yaml
ntp_enabled: True
ntp_timezone: Etc/UTC

ntp_servers:
  - "time.nist.gov iburst"
  - "ntp1.npl.co.uk iburst"
  - "ntp.sydney.nmi.gov.au iburst"
  - "time.google.com iburst"

ntp_restrict:
  - "127.0.0.1"
  - "::1"
```

## Example Playbook

```yaml
- hosts: all
  vars_files:
    - vars/main.yml
  roles:
     - { role: sparknsh.ntp }
```
